# 💰 টাকা কোথায় গেল? v3.0 AI

### প্রতিটি টাকার হিসাব রাখুন

টাকা কোথায় গেল? v3.0 AI একটি আধুনিক বাংলা Expense Tracking Web Application যা বিশেষভাবে শিক্ষার্থী, হোস্টেল/মেসে থাকা ছাত্রছাত্রী, টিউশন শিক্ষক এবং তরুণ পেশাজীবীদের জন্য তৈরি করা হয়েছে।
